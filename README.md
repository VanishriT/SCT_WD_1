# NexStream - Modern Landing Page
A sleek, responsive, and modern landing page built with HTML, CSS, and Vanilla JavaScript. NexStream is designed with a focus on modern UI/UX principles, featuring a glassmorphism aesthetic, smooth scrolling, and dynamic elements to engage users.

✨ Features
Responsive Design: Fully adaptable to mobile, tablet, and desktop screens.

Glassmorphism UI: Elegant frosted glass effects on cards and components (.glass-card).

Animated Hero Section: Eye-catching animated background gradient in the hero area.

Sticky Navigation: The navbar becomes sticky and slightly translucent as the user scrolls down.

Mobile Menu: A functional hamburger menu for seamless navigation on smaller devices.

Clean Typography: Uses the 'Poppins' font family from Google Fonts for a modern look.

Iconography: Integrated with Font Awesome for crisp, scalable vector icons.

🛠️ Technologies Used
HTML5: Semantic markup structure.

CSS3: Styling, CSS Variables for easy theming, Flexbox & CSS Grid for layouts, and Keyframe animations.

Vanilla JavaScript: Handles DOM manipulation for the sticky navbar and mobile menu toggling.

Google Fonts: Poppins

Font Awesome: Free Icons
